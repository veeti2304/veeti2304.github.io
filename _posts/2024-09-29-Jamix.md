# Mistä on kyse?
Tässä artikkelissa esittelen, kuinka voit hakea Jamix-ruokalistan Pythonilla käyttäen luomaani kirjastoa sekä Jamixin tarjoamaa rajapintaa.<br>
Se on helppoa ja nopeaa, joten jatketaanpa!

<br>

## Mistä kirjasto löytyy?
Kirjasto löytyy [GitHubista](https://github.com/veeti2304/ruokalista) 

<br>

## Kuinka kirjasto toimii?
Käytännössä kirjasto käyttää `requests`-kirjastoa hakeakseen ruokalistan ja `json`-kirjastoa muuntaakseen sen JSON-muotoon.<br>
Jos halutaan käyttää kirjastoa ja hakea esimerkiksi Gradian ruokalista se onnistuisi näin:
```python
import jamixapi # Tuodaan kirjasto

pvm = "20240123" # Päivämäärä
kId = "96786" # Keittiö-id
rId = "10" # Ruokalista-id

print(jamixapi.haeRuokaLista(pvm, kId, rId)) # Haetaan ruokalista jossa:
```
<br>
Jolloin tuloksena on JSON-muotoinen ruokalista valitsemalle päivämäärälle, keittiölle ja ruokalistalle.<br>
Keittiöiden sekä ruokalistojen id:t löytyvät kun niitä tarpeeksi etsii.<br>

Itse löysin Gradian id:t tätä kautta: 
```
https://fi.jamix.cloud/apps/menu/?anro=96786
```
Jossa anro parametri on keittiön id.
